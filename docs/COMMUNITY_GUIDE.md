# 社区建设指南 - OpenFinAgent

**目标**: 建立活跃的开源社区，促进用户交流和贡献

---

## 🌐 社区平台

### 1. GitHub (主阵地)
- **Issues**: 问题反馈和功能建议
- **Discussions**: 技术讨论和问答
- **Pull Requests**: 代码贡献

### 2. Discord 服务器 (实时交流)
- **频道结构**:
  - #welcome - 新人报到
  - #announcements - 官方公告
  - #general - 一般讨论
  - #help - 求助问答
  - #strategies - 策略分享
  - #showcase - 项目展示
  - #off-topic - 闲聊

### 3. 微信群 (中文用户)
- **群名**: OpenFinAgent 量化交易交流
- **人数上限**: 500 人
- **管理**: 2-3 名管理员
- **规则**: 禁止广告，友好交流

### 4. 知乎专栏
- **内容**: 技术文章、教程、案例分析
- **频率**: 每周 1-2 篇
- **目标**: 建立技术影响力

### 5. B 站账号
- **内容**: 视频教程、直播演示
- **频率**: 每两周 1 个视频
- **目标**: 视频化教学内容

---

## 📋 社区规则

### 通用规则
1. 尊重他人，友好交流
2. 禁止广告和 spam
3. 提问前先搜索
4. 分享知识和经验
5. 遵守开源协议

### GitHub 贡献指南
1. Fork 项目
2. 创建功能分支
3. 提交更改
4. 推送到分支
5. 创建 Pull Request

## 🏆 贡献者计划

### 等级体系
| 等级 | 要求 | 权益 |
|------|------|------|
| 👶 新手 | 注册 GitHub | 基础支持 |
| 🌱 贡献者 | 1 次 PR 合并 | 贡献者徽章 |
| 🌟 活跃贡献者 | 5 次 PR 合并 | 专属 Discord 角色 |
| 💎 核心贡献者 | 10 次 PR 合并 | 决策参与权 |
| 🚀 维护者 | 持续贡献 | 项目维护者身份 |

### 奖励机制
- GitHub Sponsors 支持
- 专属周边 (T 恤、贴纸)
- 会议演讲机会
- 推荐信
