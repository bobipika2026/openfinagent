# OpenFinAgent v0.3.2 发布说明

**发布日期**: 2026-03-07  
**版本**: v0.3.2  
**GitHub**: https://github.com/bobipika2026/openfinagent/releases/tag/v0.3.2

---

## 🎉 重大发布

OpenFinAgent v0.3.2 是一个完整的量化交易平台，从零到一打造，包含 8 种交易策略、7 个数据源、遗传算法优化器和完整的 Web UI。

---

## 🚀 新增功能

### 1. 交易策略引擎 (8 种策略)

| 策略 | 类型 | 描述 |
|------|------|------|
| 双均线交叉 | 趋势跟踪 | 短期均线上穿长期均线时买入 |
| RSI 超买超卖 | 均值回归 | RSI<30 买入，RSI>70 卖出 |
| MACD 趋势 | 趋势跟踪 | MACD 金叉买入，死叉卖出 |
| 布林带均值回归 | 均值回归 | 价格触及下轨买入，上轨卖出 |
| 动量策略 | 动量 | 追涨杀跌，动量延续 |
| 均值回归策略 | 均值回归 | 价格偏离均值时反向操作 |
| 遗传算法优化策略 | 优化 | 自动优化策略参数 |
| 自定义策略框架 | 扩展 | 用户自定义策略 |

### 2. 多数据源支持 (7 个数据源)

- **Tushare** - A 股市场数据
- **Binance** - 加密货币数据
- **Alpha Vantage** - 美股数据
- **Yahoo Finance** - 全球市场数据
- **模拟数据生成器** - 回测测试
- **CSV 数据导入** - 本地数据
- **多数据源管理器** - 统一接口

### 3. 遗传算法优化器

- 自动优化策略参数
- 参数热力图可视化
- 支持多目标优化
- 并行计算加速

### 4. 优化回测引擎

- **回测速度提升 5.24 倍**
- **内存优化 263 倍**
- 支持向量化回测
- 支持事件驱动回测

### 5. 用户系统

- JWT 认证
- RBAC 权限控制
- 用户数据模型
- 权限管理系统

### 6. 模拟交易引擎

- 模拟账户管理
- 实时交易执行
- 持仓跟踪
- 盈亏计算

### 7. Web UI

- 5 个核心页面
- 8 个可复用组件
- 实时数据可视化
- 响应式设计

页面列表:
- 仪表盘
- 策略管理
- 回测分析
- 模拟交易
- 设置

### 8. 文档站点

- 23 个文档页面
- 快速入门指南
- API 文档
- 示例代码
- 贡献指南

## 📦 安装

```bash
# 克隆仓库
git clone https://github.com/bobipika2026/openfinagent.git
cd openfinagent

# 安装依赖
pip install -e .

# 运行测试
pytest tests/

# 启动 Web UI
streamlit run openfinagent/ui/app.py
```
