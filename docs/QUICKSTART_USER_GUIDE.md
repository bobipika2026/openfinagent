## 🚀 5 分钟快速开始

### 1️⃣ 访问 Web UI

打开浏览器，访问：http://106.52.165.56:8501

---

### 2️⃣ 创建第一个策略

1. 点击左侧导航栏的 **"🤖 策略工厂"**
2. 在策略描述框中输入：
   ```
   当 5 日均线上穿 20 日均线时买入，下穿时卖出
   ```
3. 点击 **"🚀 创建策略"** 按钮
4. 等待 AI 生成策略代码（约 3 秒）
5. 查看策略预览和代码

✅ **成功标志**: 显示"策略创建成功！"提示

---

### 3️⃣ 查看策略库

1. 策略会自动保存到策略库
2. 在"策略预览"标签页查看
3. 策略状态：**草稿** (回测功能修复中)

---

### 4️⃣ 使用选股引擎

1. 点击左侧导航栏的 **"📈 选股引擎"**
2. 选择一个策略
3. 配置选股条件：
   - 市场：A 股
   - 板块：全部
   - 行业：全部
4. 点击 **"🔍 批量回测"**
5. 查看回测结果和股票池

✅ **成功标志**: 显示回测进度和结果

---

### 5️⃣ 设置监控任务

1. 点击左侧导航栏的 **"🔔 监控中心"**
2. 点击 **"新建任务"**
3. 填写任务信息：
   - 任务名称：茅台监控
   - 选择策略
   - 选择股票池
   - 监控间隔：60 秒
4. 配置通知渠道（飞书/邮件）
5. 点击 **"启动任务"**

✅ **成功标志**: 任务状态显示为"运行中"

## 💡 使用技巧

### 策略描述模板

**均线交叉**:
```
当 X 日均线上穿 Y 日均线时买入，下穿时卖出
```
示例：当 5 日均线上穿 20 日均线时买入，下穿时卖出

**动量策略**:
```
当 X 日收益率超过 Y% 时买入，低于 -Y% 时卖出
```
示例：当 20 日收益率超过 5% 时买入，低于 -5% 时卖出

**RSI 策略**:
```
当 RSI 低于 X 时买入（超卖），高于 Y 时卖出（超买）
```
示例：当 RSI 低于 30 时买入，高于 70 时卖出

**布林带策略**:
```
当价格触及下轨时买入（超卖），触及上轨时卖出（超买）
```

## 🔧 常见问题

### Q1: 策略创建失败？
**A**: 检查策略描述是否清晰，包含买入和卖出条件

### Q2: 页面加载慢？
**A**: 首次加载需要初始化模块，约 10-15 秒，后续会很快

### Q3: 数据源报错？
**A**: AkShare 数据源偶尔会限流，稍后重试即可

### Q4: 监控任务不运行？
**A**: 检查是否选择了策略和股票池
