## 步骤 1: 理解策略逻辑

### 什么是均线交叉策略？

均线交叉是最经典的量化策略之一，逻辑简单易懂：

> **策略原理**:
> - **短期均线**（如 5 日）代表近期价格趋势
> - **长期均线**（如 20 日）代表长期价格趋势
> - 当短期趋势**强于**长期趋势时 → **买入**（金叉）
> - 当短期趋势**弱于**长期趋势时 → **卖出**（死叉）

### 可视化理解

```
价格
  ↑
  |        ╱╲      买入点（金叉）
  |       ╱  ╲    ↓
  |      ╱    ╲  ╱╲
  |     ╱      ╱  ╲    卖出点（死叉）
  |    ╱      ╱    ╲  ↓
  |___╱______╱______╲______→ 时间
       5 日均线
       20 日均线
```

### 我们的策略规则

```
当 5 日均线上穿 20 日均线时买入
当 5 日均线下穿 20 日均线时卖出
初始资金 10 万元
```

## 步骤 2: 编写代码

### 创建文件

在你的工作目录创建文件 `my_first_strategy.py`：

```bash
# 创建文件
touch my_first_strategy.py

# 或使用编辑器
code my_first_strategy.py  # VS Code
vim my_first_strategy.py   # Vim
```

### 编写代码

将以下代码复制到文件中：

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
我的第一个量化策略 - 均线交叉策略
"""

from tradeflow import StrategyBuilder, BacktestEngine
from tradeflow.backtest.engine import load_data


def main():
    print("=" * 60)
    print("🚀 我的第一个量化策略")
    print("=" * 60)

    # ========== 步骤 1: 创建策略 ==========
    print("\n📝 步骤 1: 用自然语言创建策略")
    print("-" * 60)
    
    strategy = StrategyBuilder.from_natural_language("""
    当 5 日均线上穿 20 日均线时买入
    当 5 日均线下穿 20 日均线时卖出
    初始资金 10 万元
    """)
    
    print(f"✅ 策略创建成功！")
    print(f"   策略名称：{strategy.name}")
    print(f"   短期均线：{strategy.short_window}日")
    print(f"   长期均线：{strategy.long_window}日")
    print(f"   初始资金：¥{strategy.initial_capital:,.0f}")

    # ========== 步骤 2: 加载数据 ==========
    print("\n📊 步骤 2: 加载市场数据")
    print("-" * 60)
    
    data = load_data(
        symbol="600519.SH",           # 贵州茅台
        start_date="2023-01-01",
        end_date="2023-12-31",
        source='mock'                 # 使用模拟数据（无需 API）
    )
    
    print(f"✅ 数据加载完成！")
    print(f"   时间范围：{data.index[0].date()} 至 {data.index[-1].date()}")
    print(f"   数据条数：{len(data)}")
    print(f"   价格区间：¥{data['close'].min():.2f} - ¥{data['close'].max():.2f}")

    # ========== 步骤 3: 运行回测 ==========
    print("\n🔄 步骤 3: 运行回测")
    print("-" * 60)
    
    engine = BacktestEngine(
        commission_rate=0.0003,  # 手续费率：万分之三
        slippage=0.001           # 滑点：0.1%
    )
    
    results = engine.run(strategy, data, show_progress=True)
    
    print(f"✅ 回测完成！")

    # ========== 步骤 4: 显示结果 ==========
    print("\n📈 步骤 4: 回测结果")
    print("-" * 60)
    
    results.show()

    # ========== 步骤 5: 生成图表 ==========
    print("\n🎨 步骤 5: 生成图表")
    print("-" * 60)
    
    try:
        results.plot(save_path='my_first_backtest.png')
        print(f"✅ 图表已保存：my_first_backtest.png")
    except Exception as e:
        print(f"⚠️  绘图失败：{e}")
        print("   请确保已安装 matplotlib: pip install matplotlib")

    # ========== 步骤 6: 保存交易记录 ==========
    print("\n💾 步骤 6: 保存交易记录")
    print("-" * 60)
    
    results.to_csv('my_trades.csv')
    print(f"✅ 交易记录已保存：my_trades.csv")

    print("\n" + "=" * 60)
    print("🎉 恭喜！你的第一个量化策略回测完成！")
    print("=" * 60)


if __name__ == "__main__":
    main()
```

## 步骤 4: 分析结果

### 关键指标解读

#### 1. 总收益 15.23%

- **含义**: 策略在回测期间总共盈利 15.23%
- **评价**: 跑赢银行理财，但需要结合风险看

#### 2. 年化收益 15.23%

- **含义**: 如果每年都能获得同样收益，年化回报为 15.23%
- **评价**: 高于股市平均回报（约 8-10%）

#### 3. 夏普比率 0.83

- **含义**: 每承担 1 单位风险，获得 0.83 单位的超额收益
- **评价**: 
  - > 1.0：优秀
  - 0.5-1.0：良好
  - < 0.5：较差
- **当前**: 0.83 属于良好水平

#### 4. 最大回撤 -12.34%

- **含义**: 策略历史上最大的亏损幅度
- **评价**: 
  - < 10%：低风险
  - 10-20%：中等风险
  - > 20%：高风险
- **当前**: -12.34% 属于中等风险

#### 5. 胜率 58.33%

- **含义**: 58.33% 的交易是盈利的
- **评价**: 超过 50% 说明策略逻辑有效

## 步骤 5: 优化策略

### 尝试不同的均线组合

修改代码中的策略参数：

```python
# 方案 1: 更敏感的策略（交易更频繁）
strategy = StrategyBuilder.create(
    'ma_cross',
    short_window=3,      # 缩短短期均线
    long_window=10,      # 缩短长期均线
    initial_capital=100000
)

# 方案 2: 更稳健的策略（交易更少）
strategy = StrategyBuilder.create(
    'ma_cross',
    short_window=20,     # 延长短期均线
    long_window=60,      # 延长长期均线
    initial_capital=100000
)

# 方案 3: 经典组合
strategy = StrategyBuilder.create(
    'ma_cross',
    short_window=10,
    long_window=30,
    initial_capital=100000
)
```

### 添加风险控制

```python
strategy = StrategyBuilder.from_natural_language("""
当 5 日均线上穿 20 日均线时买入
当 5 日均线下穿 20 日均线时卖出
初始资金 10 万元
止损 10%
止盈 30%
最大仓位 80%
""")
```

### 对比不同参数

```python
from tradeflow import StrategyBuilder, BacktestEngine

# 创建多个策略
strategies = {
    '激进型 (3/10)': StrategyBuilder.create('ma_cross', short_window=3, long_window=10),
    '标准型 (5/20)': StrategyBuilder.create('ma_cross', short_window=5, long_window=20),
    '稳健型 (10/30)': StrategyBuilder.create('ma_cross', short_window=10, long_window=30),
    '保守型 (20/60)': StrategyBuilder.create('ma_cross', short_window=20, long_window=60),
}

# 批量回测
engine = BacktestEngine()
results = engine.run_multi(strategies, data)

# 对比结果
print("\n策略对比:")
print("-" * 60)
for name, result in results.items():
    metrics = result.metrics
    print(f"{name:15} 年化：{metrics['年化收益']:6.2f}%  夏普：{metrics['夏普比率']:.2f}  回撤：{metrics['最大回撤']:6.2f}%")
```

## 💡 小贴士

1. **从简单开始**: 先用默认参数，再逐步优化
2. **多周期测试**: 在不同时间段测试策略（牛市、熊市、震荡市）
3. **关注风险**: 收益重要，但风险控制更重要
4. **记录实验**: 保存每次回测的参数和结果，方便对比
5. **避免过拟合**: 不要在单一股票上过度优化参数

## ❓ 常见问题

### Q: 为什么使用模拟数据？

A: 模拟数据无需 API 密钥，适合学习和测试。熟悉后可以使用真实数据：

```python
# A 股真实数据
data = load_data("600519.SH", "2023-01-01", "2023-12-31", source='akshare')

# 美股真实数据
data = load_data("AAPL", "2023-01-01", "2023-12-31", source='yfinance')
```

### Q: 回测结果和实盘不一样？

A: 这是正常的，原因包括：
- 回测是历史数据，实盘是未来数据
- 实盘有冲击成本、流动性问题
- 策略可能过拟合历史数据

**建议**: 回测只是第一步，实盘前需要更多验证。

### Q: 如何提高策略收益？

A: 可以尝试：
1. 优化均线周期
2. 添加过滤条件（如成交量、RSI）
3. 动态调整仓位
4. 结合多个策略
