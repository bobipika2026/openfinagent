## 基础语法

OpenFinAgent 使用**自然语言解析技术**，将中文策略描述转换为可执行的策略代码。

### 基本结构

一个完整的策略描述包含以下部分：

```
[策略类型]
[交易规则]
[资金管理]
[风险控制]
```

### 示例

```
均线交叉策略
当 5 日均线上穿 20 日均线时买入
当 5 日均线下穿 20 日均线时卖出
初始资金 10 万元
```

## 策略类型

OpenFinAgent 支持以下策略类型：

### 1. 均线交叉策略 (MA Cross)

**语法格式**:
```
均线交叉策略
当 [短期] 日均线上穿 [长期] 日均线时买入
当 [短期] 日均线下穿 [长期] 日均线时卖出
```

**示例**:
```
当 5 日均线上穿 20 日均线时买入
当 5 日均线下穿 20 日均线时卖出
初始资金 10 万元
```

**参数说明**:
- 短期均线：通常为 5、10、20 日
- 长期均线：通常为 20、30、60 日

### 2. 动量策略 (Momentum)

**语法格式**:
```
动量策略
看 [周期] 天收益率
超过 [阈值]% 买入
低于 [阈值]% 卖出
```

**示例**:
```
动量策略
看 20 天收益率
超过 5% 买入
低于 -5% 卖出
初始资金 50 万元
```

**参数说明**:
- 周期：计算收益率的时间窗口（默认 20 天）
- 阈值：触发交易的收益率阈值

### 3. 自定义策略

**语法格式**:
```
自定义策略
[你的交易规则]
```

**示例**:
```
自定义策略
当收盘价高于 30 日均线时持有
当收盘价低于 30 日均线时空仓
初始资金 20 万元
```

## 参数说明

### 资金参数

| 参数 | 语法 | 示例 | 说明 |
|------|------|------|------|
| 初始资金 | `初始资金 [金额]` | `初始资金 10 万元` | 策略启动资金 |
| 单笔最大 | `单笔最大 [百分比]%` | `单笔最大 20%` | 单笔交易最大仓位 |
| 最大仓位 | `最大仓位 [百分比]%` | `最大仓位 80%` | 总仓位上限 |

### 交易参数

| 参数 | 语法 | 示例 | 说明 |
|------|------|------|------|
| 手续费率 | `手续费 [万分之 X]` | `手续费 万分之三` | 交易手续费（默认万三） |
| 滑点 | `滑点 [百分比]%` | `滑点 0.1%` | 价格滑点（默认 0.1%） |
| 最小交易单位 | `最小 [股数] 股` | `最小 100 股` | A 股最小交易单位（默认 100） |

### 风险参数

| 参数 | 语法 | 示例 | 说明 |
|------|------|------|------|
| 止损 | `止损 [百分比]%` | `止损 10%` | 单笔下跌幅度止损 |
| 止盈 | `止盈 [百分比]%` | `止盈 20%` | 单笔上涨幅度止盈 |
| 最大回撤 | `最大回撤 [百分比]%` | `最大回撤 15%` | 策略整体回撤止损 |

## 高级用法

### 组合条件

支持多个条件组合：

```
当 5 日均线上穿 20 日均线 且 RSI < 30 时买入
当 5 日均线下穿 20 日均线 或 RSI > 70 时卖出
```

### 时间过滤

限制交易时间段：

```
只在 9:30-14:30 交易
当 5 日均线上穿 20 日均线时买入
当 5 日均线下穿 20 日均线时卖出
```

### 仓位管理

动态仓位控制：

```
当 5 日均线上穿 20 日均线时买入 50% 仓位
当 5 日均线下穿 20 日均线时卖出全部仓位
```

### 多条件策略

```
高级策略
条件 1: 5 日均线上穿 20 日均线
条件 2: 成交量放大 20%
条件 3: RSI < 40
当所有条件满足时买入
当 5 日均线下穿 20 日均线时卖出
初始资金 50 万元
止损 10%
止盈 30%
```

## 代码方式创建策略

除了自然语言，也可以使用参数化方式：

### 均线交叉策略

```python
strategy = StrategyBuilder.create(
    'ma_cross',
    short_window=5,      # 短期均线周期
    long_window=20,      # 长期均线周期
    initial_capital=100000,  # 初始资金
    commission_rate=0.0003,  # 手续费率
    name='MA5/20 策略'
)
```

### 动量策略

```python
strategy = StrategyBuilder.create(
    'momentum',
    lookback=20,         # 回看周期
    threshold=0.05,      # 阈值 5%
    initial_capital=500000,
    stop_loss=0.08,      # 止损 8%
    take_profit=0.25,    # 止盈 25%
    name='动量突破策略'
)
```

## 常见问题

### Q: 策略描述有格式要求吗？

A: 尽量使用简洁清晰的中文，按照"条件 + 动作"的格式描述。系统会自动识别关键词。

### Q: 支持哪些技术指标？

A: 当前支持：
- 均线 (MA)
- 相对强弱指标 (RSI)
- 成交量 (Volume)

更多指标将在后续版本中添加。

### Q: 如何调试策略？

A: 使用以下方法：

```python
# 查看解析后的策略参数
print(strategy.get_params())

# 查看策略详情
print(strategy.summary())
```

### Q: 自然语言解析不准确怎么办？

A: 可以尝试：
1. 使用更简洁的描述
2. 使用参数化方式 `StrategyBuilder.create()`
3. 继承 `BaseStrategy` 自定义策略

### Q: 支持股票以外的品种吗？

A: 支持：
- A 股（600519.SH）
- 美股（AAPL）
- 加密货币（BTC/USDT）

使用 `load_data()` 时指定正确的代码格式即可。

## 最佳实践

1. **从简单开始**: 先用基础策略验证想法
2. **明确参数**: 清晰描述均线周期、阈值等参数
3. **风险控制**: 设置合理的止损止盈
4. **充分回测**: 使用至少 1 年数据验证
5. **记录实验**: 保存每次回测的参数和结果
