## 新手入门案例

### 案例 1: 第一个均线策略

**用户**: @quant_newbie  
**策略类型**: 均线交叉  
**回测周期**: 2023 年全年  
**标的**: 贵州茅台 (600519.SH)

#### 策略描述

```
当 5 日均线上穿 20 日均线时买入
当 5 日均线下穿 20 日均线时卖出
初始资金 10 万元
```

#### 回测结果

| 指标 | 数值 |
|------|------|
| 总收益 | +15.23% |
| 年化收益 | +15.23% |
| 夏普比率 | 0.83 |
| 最大回撤 | -12.34% |
| 交易次数 | 24 |
| 胜率 | 58.33% |

#### 用户心得

> "作为量化新手，OpenFinAgent 让我第一次体验到了用中文写策略的便利。不需要学习复杂的编程语法，只需要描述我的交易想法，系统就能自动生成策略并回测。"
> 
> "第一次看到回测结果时很兴奋，虽然收益不是特别高，但验证了我的想法是可行的。接下来我打算尝试优化均线周期，看看能不能提高收益。"

#### 代码实现

```python
from openfinagent import StrategyBuilder, BacktestEngine
from openfinagent.backtest.engine import load_data

strategy = StrategyBuilder.from_natural_language("""
当 5 日均线上穿 20 日均线时买入
当 5 日均线下穿 20 日均线时卖出
初始资金 10 万元
""")

data = load_data("600519.SH", "2023-01-01", "2023-12-31", source='akshare')
engine = BacktestEngine()
results = engine.run(strategy, data)

results.show()
results.plot()
```

### 案例 4: 行业轮动策略

**用户**: @sector_rotation  
**策略类型**: 自定义策略  
**回测周期**: 2021-2023 年  
**标的**: 多个行业 ETF

#### 策略逻辑

```python
from openfinagent.strategy import BaseStrategy

class SectorRotationStrategy(BaseStrategy):
    """
    行业轮动策略：
    1. 计算各行业 ETF 的 20 日动量
    2. 买入动量最强的 3 个行业
    3. 每周调仓一次
    """
    
    def on_bar(self, data, index):
        # 计算动量
        momentum = data['close'].pct_change(20)
        
        # 选择最强的 3 个行业
        top_sectors = momentum.nlargest(3).index
        
        # 生成交易信号
        if self.current_sector not in top_sectors:
            return {'action': 'rebalance', 'sectors': top_sectors}
        
        return {'action': 'hold'}
```

#### 回测结果

| 指标 | 数值 |
|------|------|
| 总收益 | +68.92% |
| 年化收益 | +22.97% |
| 夏普比率 | 1.45 |
| 最大回撤 | -18.23% |
| 交易次数 | 104 |
| 胜率 | 65.38% |

#### 用户心得

> "作为有一定经验的量化交易者，我很欣赏 OpenFinAgent 的扩展性。内置策略适合快速验证想法，自定义策略则给了我完全的控制权。"

### 案例 6: 美股科技股策略

**用户**: @us_tech_trader  
**策略类型**: 动量 + 趋势  
**回测周期**: 2020-2023 年  
**标的**: 纳斯达克 100 成分股

#### 策略实现

```python
from openfinagent import StrategyBuilder, BacktestEngine
import yfinance as yf

# 创建策略
strategy = StrategyBuilder.create(
    'momentum',
    lookback=30,
    threshold=0.08,
    initial_capital=500000,
    stop_loss=0.12,
    take_profit=0.35
)

# 加载美股数据
tickers = ['AAPL', 'MSFT', 'GOOGL', 'AMZN', 'NVDA']
for ticker in tickers:
    data = load_data(ticker, "2020-01-01", "2023-12-31", source='yfinance')
    results = engine.run(strategy, data)
    print(f"{ticker}: 年化收益 {results.metrics['年化收益']:.2f}%")
```

#### 回测结果汇总

| 标的 | 年化收益 | 夏普比率 | 最大回撤 |

| AAPL | +28.5% | 1.52 | -21.3% |
| MSFT | +24.8% | 1.38 | -18.7% |
| GOOGL | +19.2% | 1.15 | -25.4% |
| AMZN | +15.6% | 0.98 | -32.1% |
| NVDA | +45.3% | 1.78 | -28.9% |

#### 用户心得
