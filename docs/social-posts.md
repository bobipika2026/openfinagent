## 📝 知乎

### 标题
如何评价 OpenFinAgent v0.3.2 开源量化交易平台？

### 正文
今天，OpenFinAgent v0.3.2 正式发布，这是一个从零到一打造的完整量化交易平台。

**核心数据**：
- 开发时间：~150 分钟
- AI 代理数：18 个
- 代码行数：12,088 行
- 测试覆盖率：73.7%
- 文档页数：23 页

**功能亮点**：

1. **8 种交易策略**
   覆盖趋势跟踪、均值回归、动量等多种交易理念，包括双均线交叉、RSI 超买超卖、MACD 趋势、布林带均值回归等经典策略。

2. **7 个数据源**
   支持 Tushare(A 股)、Binance(加密货币)、Alpha Vantage(美股)、Yahoo Finance 等，满足全球市场数据需求。

3. **遗传算法优化器**
   自动优化策略参数，支持参数热力图可视化，回测速度提升 5.24 倍，内存优化 263 倍。

4. **完整 Web UI**
   基于 Streamlit 构建，包含仪表盘、策略管理、回测分析、模拟交易、设置等 5 个核心页面。

**技术栈**：
- Python 3.10+
- NumPy/Pandas
- TA-Lib
- Streamlit
- Pytest
- JWT

**快速开始**：
```bash
git clone https://github.com/bobipika2026/openfinagent.git
cd openfinagent
pip install -e .
streamlit run openfinagent/ui/app.py
```

**项目愿景**：
用 AI 降低量化门槛，用体验赢得用户，用生态建立壁垒。

GitHub: https://github.com/bobipika2026/openfinagent

欢迎 Star、Fork、提 Issue！🌟
