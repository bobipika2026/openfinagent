### 1. Binance 数据源 - 加密货币交易新选择

v0.3.0 新增了 **Binance 数据源**，支持获取全球最大加密货币交易所的实时和历史数据。

**核心特性：**

- ✅ **免 API Key** - 使用公开接口，开箱即用
- ✅ **多交易对** - BTC/USDT、ETH/USDT 等所有 Binance 交易对
- ✅ **14 种时间周期** - 1m, 5m, 15m, 1h, 4h, 1d, 1w, 1M 等
- ✅ **自动分页** - 大数据量自动分批请求
- ✅ **限流控制** - 智能请求频率管理

**快速示例：**

```python

# 创建数据源

# 获取 BTC/USDT 日线数据

    symbol='BTC/USDT',
    interval='1d',
    start_date='2024-01-01',
    end_date='2024-12-31'
)

print(f"获取到 {len(data)} 行数据")
```

现在你可以轻松回测加密货币策略，捕捉数字资产市场的交易机会！

- ✅ **复权处理** - 前复权、后复权、不复权
- ✅ **Token 管理** - 安全的 Token 配置和加密存储
- ✅ **缓存机制** - 避免重复请求，提高效率

**快速示例：**


### 3. 遗传算法优化器 - 智能参数搜索

v0.3.0 最重磅的功能 - **遗传算法优化器**！

还在手动试错调整参数？让遗传算法帮你自动搜索最优参数组合吧！

**核心特性：**

- ✅ **智能搜索** - 基于生物进化原理的全局优化
- ✅ **多种群进化** - 选择、交叉、变异操作
- ✅ **精英保留** - 保留最优个体到下一代
- ✅ **并行计算** - 多进程并行评估，加速优化
- ✅ **灵活配置** - 自定义优化目标和约束

**快速示例：**

```python
from optimization.genetic_optimizer import GeneticOptimizer, ParameterBound
from backtest.engine import BacktestEngine
from strategy.dual_ma import DualMAStrategy

# 定义参数范围
param_bounds = [
    ParameterBound('short_window', 5, 20, dtype='int'),
    ParameterBound('long_window', 20, 60, dtype='int'),
]

# 创建优化器
optimizer = GeneticOptimizer(
    population_size=50,
    generations=100,
    optimization_metric='sharpe_ratio',
    n_jobs=-1  # 使用所有 CPU 核心
)

# 执行优化
best = optimizer.optimize(
    strategy_class=DualMAStrategy,
    parameter_bounds=param_bounds,
    backtest_func=lambda s, d: BacktestEngine().run(s, d),
    data=data,
    initial_capital=100000
)

print(f"最优参数：{best.params}")
print(f"夏普比率：{best.fitness:.2f}")
```

**算法流程：**

```
初始化种群 (50 个个体)
     ↓
评估适应度 (回测每个个体)
     ↓
选择优秀个体 (锦标赛选择)
     ↓
交叉产生子代 (80% 概率)
     ↓
变异增加多样性 (10% 概率)
     ↓
精英保留 (前 10%)
     ↓
新一代种群
     ↓
循环 100 代
     ↓
输出最优参数
```

### 4. 参数热力图 - 可视化参数敏感性分析

除了遗传算法，我们还提供了**参数热力图**工具，用于双参数敏感性分析。

**核心特性：**

- ✅ **双参数扫描** - 全面分析两个参数的影响
- ✅ **热力图展示** - 直观显示参数敏感性
- ✅ **最优参数标记** - 自动标注最佳组合
- ✅ **交互式图表** - 支持 Plotly 交互可视化
- ✅ **等高线图** - 专业的参数空间分析

**快速示例：**

```python
from visualization.heatmap import ParameterHeatmap
from backtest.engine import BacktestEngine

# 创建热力图生成器
heatmap = ParameterHeatmap(metric_name='sharpe_ratio')

# 执行双参数扫描
result = heatmap.scan(
    strategy_class=DualMAStrategy,
    param1_name='short_window',
    param1_values=[5, 10, 15, 20],
    param2_name='long_window',
    param2_values=[20, 30, 40, 50, 60],
    base_params={},
    backtest_func=lambda s, d: BacktestEngine().run(s, d),
    data=data,
    initial_capital=100000
)

# 绘制热力图
heatmap.plot(
    result,
    title='双均线策略参数热力图',
    save_path='heatmap.png'
)

print(f"最优参数：short_window={result.best_param1}, long_window={result.best_param2}")
```

**可视化效果：**

![参数热力图示例](/assets/images/heatmap-example.png)

通过热力图，你可以直观地看到：
- 哪些参数组合表现最好
- 参数敏感性如何（陡峭 vs 平坦）
- 是否存在过拟合风险（孤立的最优点）

## 🚀 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/bobipika2026/openfinagent.git
cd openfinagent

# 安装核心依赖
pip install -r requirements.txt

# 可选：加密货币数据
pip install requests

# 可选：A 股分钟线
pip install tushare

# 可选：交互式图表
pip install plotly
```

### 完整工作流示例

```python
# 1. 获取数据（Binance 或 Tushare）
from data.binance_source import BinanceDataSource

source = BinanceDataSource()
data = source.get_klines('BTC/USDT', '1d', '2023-01-01', '2023-12-31')

# 2. 定义策略
from strategy.dual_ma import DualMAStrategy

# 3. 参数优化
from optimization.genetic_optimizer import GeneticOptimizer, ParameterBound
from backtest.engine import BacktestEngine

param_bounds = [
    ParameterBound('short_window', 5, 20, dtype='int'),
    ParameterBound('long_window', 20, 60, dtype='int'),
]

optimizer = GeneticOptimizer(population_size=30, generations=50)
best = optimizer.optimize(
    strategy_class=DualMAStrategy,
    parameter_bounds=param_bounds,
    backtest_func=lambda s, d: BacktestEngine().run(s, d),
    data=data
)

# 4. 使用最优参数回测
strategy = DualMAStrategy(**best.params, initial_capital=10000)
result = BacktestEngine().run(strategy, data)

# 5. 查看结果
print(result.summary())
result.plot()
```

## ⚠️ 注意事项

### 1. Tushare Token

使用 Tushare 分钟线需要 API token：

- 注册地址：https://tushare.pro
- 分钟线需要 2000 积分
- 设置环境变量：`export TUSHARE_TOKEN=your_token`

### 2. 网络要求

Binance 数据源需要访问国际互联网：

```python
# 检查网络连接
import requests
try:
    response = requests.get('https://api.binance.com', timeout=5)
    print("网络连接正常")
except:
    print("无法访问 Binance API")
```

### 3. 性能建议

大数据集建议使用优化引擎 + 缓存：

```python
from backtest.optimized_engine import OptimizedBacktestEngine

engine = OptimizedBacktestEngine(use_cache=True)
result = engine.run(strategy, data)
```

### 4. 过拟合风险

参数优化时注意避免过拟合：

- 使用足够长的历史数据（2-3 年）
- 留出验证集进行测试
- 使用交叉验证
- 选择稳健的优化指标（如夏普比率）
