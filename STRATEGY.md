# 🌌 OpenFinAgent - 金融 AI Agent 操作系统

## 使命宣言

> **让每个人拥有自己的私人投行 AI，让每个开发者都能构建金融智能体**

---

## 🔮 愿景（来自 300 年后的视角）

站在未来回望，2025 年是金融科技的"功能机时代"：
- 量化交易 = 手动写策略 + 回测 + 执行（割裂的工具链）
- 智能投顾 = 规则引擎 + 简单推荐（伪 AI）
- 风险管理 = 事后监控 + 规则告警（被动响应）
- 投研分析 = 人工阅读 + Excel 整理（低效劳动）

**真正的革命是什么？**

→ **金融 AI Agent 操作系统**：一个让 AI 智能体自主协作、自我进化、去中心化运行的开源协议层

---

## 💡 核心颠覆点

| 维度 | 现在（2025） | OpenFinAgent（革命） |
|------|-------------|---------------------|
| **交互方式** | 写代码/配置策略 | 自然语言对话，AI 理解意图 |
| **决策主体** | 人决策，机器执行 | AI Agent 自主决策 + 人监督 |
| **策略形态** | 静态代码 | 自我进化的"策略生命体" |
| **协作模式** | 单人单策略 | 多 Agent 协作网络（投研/交易/风控） |
| **数据孤岛** | 各平台数据不互通 | 统一数据协议 + 链上存证 |
| **知识积累** | 策略无法复用 | 策略市场 + 知识图谱 |
| **部署方式** | 中心化平台 | 去中心化节点网络 |

## 🎯 第一阶段 MVP（30 天冲刺）

### 核心功能
1. **自然语言策略生成** - 说话即策略
2. **多 Agent 协作框架** - 投研 + 交易 + 风控三联体
3. **策略回测引擎** - 支持 A 股/美股/加密货币
4. **GitHub 开源发布** - 完整文档 + 示例 + 社区运营

### 技术栈
- **核心框架**: Python 3.11+
- **AI 引擎**: Qwen/Llama + 金融领域微调
- **数据源**: AkShare/YFinance + 自建爬虫
- **回测引擎**: 自研（支持分钟级/ tick 级）
- **前端**: Streamlit/FastAPI
- **部署**: Docker + Kubernetes

## ⚠️ 风险控制

1. **合规风险** - 明确声明"仅供研究学习，不构成投资建议"
2. **技术风险** - 沙箱环境回测，实盘需人工确认
3. **安全风险** - API Key 加密存储，权限最小化
4. **声誉风险** - 透明披露回测局限性，避免过度承诺
