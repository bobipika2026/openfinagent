# OpenFinAgent KOL Outreach 私信模板

**版本**: v1.0  
**适用平台**: 微信、微博、知乎、B 站  
**核心原则**: 简洁、真诚、低压迫感

---

## 💬 微信私信模板

### 模板一：首次添加好友（验证消息）

```
{KOL 姓名}老师您好，我是 OpenFinAgent 项目负责人，
关注您很久，想请教量化/AI 交易相关合作，
盼通过，谢谢！
```

**要点**: 20 字以内，说明身份 + 目的 + 礼貌

---

### 模板二：通过后首次消息

```
{KOL 姓名}老师好！感谢通过 🙏

我是 OpenFinAgent（开源 AI 量化交易平台）的负责人。
一直在关注您的{具体内容方向}，特别认可您的{具体观点/文章}。

我们即将发布 v0.3.2 版本，想邀请您体验并给些专业建议。
如果感兴趣，可以发您资料看看，或者安排个 15 分钟演示。

不知您是否方便？😊
```

**要点**: 
- 先感谢，降低压迫感
- 简短介绍 + 具体认可点
- 给出低门槛选项（看资料/短演示）

---

### 模板三：发送资料后跟进

```
{KOL 姓名}老师，资料您有空可以看看~

我们特别希望能听到您的专业反馈，
如果觉得产品还不错，也可以聊聊合作可能。

预算方面我们可以根据您的报价来，
形式也灵活（评测/教程/直播都可以）。

等您消息，不着急 😊
```

**要点**: 
- 强调"反馈"而非"合作"
- 表明预算灵活
- "不着急"降低压力

---

### 模板四：KOL 表示兴趣后

```
太好了！感谢认可 🎉

那我跟您同步下合作细节：

【合作形式】{具体形式，如：公众号文章}
【内容方向】{具体方向，如：产品评测/使用教程}
【发布时间】{期望时间，如：3 月中旬}
【合作预算】¥{金额}（可协商）
【额外福利】专业版账号 + 粉丝专属优惠码

您看这个方案如何？或者您有什么想法也可以说~
```

**要点**: 
- 清晰列出合作要素
- 保持协商空间
- 表情符号增加亲和力

---

### 模板五：KOL 犹豫/考虑中

```
理解理解，确实需要时间考虑~

这样吧，我先给您开通个 3 个月的专业版账号，
您先体验看看，觉得好再考虑合作。

账号信息我稍后发您，有任何问题随时问我。
合作的事咱们不急，产品好用最重要 💪
```

**要点**: 
- 先给价值，再谈合作
- 降低决策压力
- 展现长期思维

---

### 模板六：KOL 拒绝后

```
完全理解，感谢您坦诚告知 🙏

那我先不打扰您了，如果后续有合作想法随时联系我。
账号还是给您留着，当个朋友，有空多交流~

祝内容创作顺利！📈
```

**要点**: 
- 优雅接受拒绝
- 保持关系（未来可能合作）
- 留下好印象

## 📱 各平台私信最佳实践

### 微信
- **优势**: 即时沟通，关系更近
- **注意**: 先加好友验证，通过后别急着发广告
- **节奏**: 验证→感谢→介绍→资料→跟进

### 微博
- **优势**: 公开互动建立初步联系
- **注意**: 私信有字数限制，引导加微信
- **节奏**: 评论互动→私信→微信

### 知乎
- **优势**: 专业氛围，KOL 更开放
- **注意**: 以"请教"姿态切入
- **节奏**: 认可内容→请教问题→合作邀请

### B 站
- **优势**: 视频内容更直观
- **注意**: 先支持（三连）再联系
- **节奏**: 观看视频→评论/私信→微信

## 📊 私信回复率提升技巧

### 1. 个性化
- 提及具体内容（文章/视频/观点）
- 使用 KOL 的名字/昵称
- 证明你不是群发

### 2. 低门槛
- 先给价值（资料/体验账号）
- 不急于谈合作
- 提供多个选项

### 3. 真诚
- 表达真实认可
- 承认需要帮助
- 接受拒绝

### 4. 跟进节奏
- 首次无回复：3-5 天后跟进
- 二次无回复：7 天后最后确认
- 三次无回复：暂停，1 个月后再试

## 🚫 私信禁忌

- ❌ 一上来就发合作报价
- ❌ 群发痕迹明显（不提及具体内容）
- ❌ 过度吹捧（显得虚假）
- ❌ 频繁催促（每天追问）
- ❌ 被拒后纠缠
- ❌ 长篇大论（超过 500 字）
