## 🚀 快速启动

### Day 1: 准备

1. **阅读文件** (1 小时)
   - 先读 `collaboration-plan.md` 了解整体策略
   - 再读 `kol-list-complete.md` 熟悉 KOL 名单

2. **准备物料** (2 小时)
   - 准备产品介绍.pdf
   - 准备演示视频/GIF
   - 准备合作合同模板

3. **设置追踪** (30 分钟)
   - 复制 `tracking-sheet.md` 为工作表
   - 填写负责人信息
   - 设置每日提醒

### Day 2-5: 首轮联系

```
每日目标：联系 10-15 位 KOL
- 上午 10:00-11:30：发送 5-8 封邮件
- 下午 14:00-16:00：发送 5-7 条私信
- 晚上 20:00-21:00：整理回复，更新追踪表
```

**联系顺序**:
1. 优先联系 P0 级 KOL (量化领域 1-5 号)
2. 其次联系 P1 级 KOL (技术博主 1-10 号)
3. 最后联系 P2 级 KOL (金融科技 1-10 号)

### Week 2: 跟进与演示

```
- 跟进未回复 KOL (首次邮件后 3-5 天)
- 安排有意向 KOL 产品演示
- 开始合同洽谈
```

### Week 3-4: 签约与启动

```
- 目标签约 5 位 KOL
- 开通体验账号
- 确认内容大纲
```

## ⚠️ 注意事项

### 合规风险

- ❌ 避免承诺收益（"稳赚不赔"、"年化 XX%"）
- ✅ 使用客观描述（"历史回测"、"仅供参考"）
- ✅ 标注合作内容（"广告"、"合作"）

### 预算控制

- 单篇内容不超过预算的 5% (¥2,400)
- 头部 KOL 不超过总预算的 50%
- 预留 10% 备用金

### 合同要点

- 50% 预付款 +50% 发布后付款
- 约定最低曝光/注册量
- 30 天竞品排他条款
- 内容修改权利

## 📞 常用话术

### KOL 问："你们产品有什么优势？"

```
核心是三点：
1. AI 驱动 - 大模型辅助策略分析和市场判断
2. 低门槛 - 可视化搭建，无需深厚编程基础
3. 全市场 - A 股/港股/美股/期货全覆盖

跟传统量化平台比，我们更像"量化界的 Notion"，
让普通人也能轻松搭建自己的量化策略。
```

### KOL 问："合作报价多少？"

```
根据内容形式和粉丝量级，一般在¥{范围}之间。
具体可以根据您的需求定制，比如：
- 轻量体验分享：¥{低}
- 深度评测教程：¥{中}
- 直播 + 视频组合：¥{高}

您平时接合作的报价大概是多少？我们可以参考。
```

### KOL 问："效果怎么保证？"

```
我们提供：
1. 完整素材包（截图/数据/案例）降低创作成本
2. 专属优惠码追踪效果
3. 实时数据看板同步
4. 效果不达标可协商补偿

更看重长期合作，单次效果不理想可以调整优化。
```

## 🎯 成功要素

1. **个性化** - 每封邮件/私信都要提及 KOL 的具体内容
2. **低门槛** - 先给体验账号，再谈合作
3. **快响应** - KOL 回复后 24 小时内安排演示
4. **真体验** - 要求 KOL 实际使用至少 7 天
5. **长思维** - 不急于单次合作，建立长期关系
