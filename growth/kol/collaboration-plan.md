## 📝 内容要求

### 基础要求（所有合作）

- ✅ 真实体验：KOL 需实际使用产品至少 7 天
- ✅ 客观评价：允许提出批评和建议
- ✅ 核心亮点：至少提及 2-3 个核心功能
- ✅ 行动号召：提供优惠码/注册链接
- ✅ 合规声明：标注"合作内容"或"广告"

### 内容方向建议

#### 技术向 KOL
- GitHub 项目深度解读
- 技术架构分析
- 代码示例/教程
- 与竞品技术对比

#### 量化向 KOL
- 策略搭建教程
- 回测结果分享
- AI 信号实测
- 与传统量化平台对比

#### 财经向 KOL
- 量化投资科普
- AI 交易趋势分析
- 用户案例故事
- 投资工具评测

### 内容审核流程

```
1. KOL 提交内容大纲 → 2. 我们确认（1 个工作日）
→ 3. KOL 创作 → 4. 提交初稿 → 5. 我们审核（1 个工作日）
→ 6. 修改完善 → 7. 最终确认 → 8. 发布
```

**审核要点**:
- 产品信息准确性
- 合规性（避免违规承诺）
- 品牌调性一致性
- 关键信息完整性

**注意**: 尊重的 KOL 的创作自由，仅审核事实准确性，不干涉观点表达
