## 📚 FAQ 文档

### FAQ 结构

```markdown
# OpenFinAgent 常见问题 (FAQ)

## 📦 安装相关

### Q1: OpenFinAgent 支持哪些操作系统？
**A**: 支持 Windows、macOS、Linux。推荐使用 Ubuntu 20.04+ 或 macOS 12+。

### Q2: 安装失败怎么办？
**A**: 
1. 确认 Python 版本 ≥ 3.10
2. 使用虚拟环境安装
3. 查看完整错误日志
4. 在 GitHub Issues 搜索类似问题

### Q3: 需要付费吗？
**A**: 完全开源免费 (MIT License)，可自由使用和二次开发。

---

## 🔧 使用相关

### Q4: 如何创建第一个交易策略？
**A**: 
```bash
# 1. 启动 OpenFinAgent
openfinagent start

# 2. 访问 Web 界面
http://localhost:8080

# 3. 点击"新建策略"，输入自然语言描述
# 例："当 5 日均线上穿 20 日均线时买入"
```

### Q5: 支持哪些券商？
**A**: 
- A 股：华泰、中信、国泰君安等 (通过通用接口)
- 美股：Interactive Brokers
- 加密货币：Binance、OKX、Huobi

### Q6: 回测数据从哪里来？
**A**: 
- 默认使用内置的免费数据源
- 支持导入本地数据
- 支持接入第三方数据服务 (Tushare、聚宽等)

---

## 🤖 AI 相关

### Q7: AI 策略生成的准确率如何？
**A**: v0.3.2 版本准确率达 92%，复杂策略可能需要手动调整。

### Q8: 使用 AI 功能需要额外付费吗？
**A**: 
- 本地模型：完全免费
- 云端 API：按使用量计费 (支持自带 API Key)

### Q9: 我的策略数据会上传吗？
**A**: 不会。所有数据处理在本地完成，除非你主动使用云端服务。

---

## 🔒 安全相关

### Q10: 实盘交易安全吗？
**A**: 
- 资金由券商托管，OpenFinAgent 只发送交易指令
- 内置多重风控：仓位限制、止损、异常检测
- 建议先用模拟盘测试

### Q11: API Key 如何存储？
**A**: 
- 本地加密存储
- 支持环境变量
- 建议使用只读/交易权限分离的 API Key
