## 📝 使用示例

### 示例 1：自然语言创建策略

```
输入：
"当 5 日均线上穿 20 日均线时买入，下穿时卖出，初始资金 10 万元"

输出：
策略名称：MA_Cross_5_20
参数：short_window=5, long_window=20, initial_capital=100000
```

### 示例 2：运行回测

```
股票：000001（平安银行）
时间：2023-01-01 至 2023-12-31
策略：均线交叉 (5,20)

结果：
- 总收益：+15.3%
- 夏普比率：1.25
- 最大回撤：-8.5%
- 交易次数：12
```

### 示例 3：策略对比

```
对比策略：
1. 均线交叉 (5,20)
2. RSI (14,30,70)
3. MACD (12,26,9)

结果：
- 最佳收益：RSI (+18.5%)
- 最低回撤：均线交叉 (-6.2%)
- 最高夏普：MACD (1.45)
```

## 🔧 修复的问题

在开发过程中修复了以下问题：

1. **模块导入路径问题**
   - 修复了 `engine.py` 中的绝对导入为相对导入
   - 添加了兼容性导入（try-except）

2. **策略参数名称不一致**
   - RSI 策略：`period` → `rsi_period`
   - RSI 策略：`oversold` → `oversold_threshold`
   - RSI 策略：`overbought` → `overbought_threshold`
   - 布林带策略：`std_dev` → `num_std`
   - 双均线策略：`volume_ma_period` → `volume_window`

3. **UI 细节问题**
   - 修复了 `st.metric` 语法错误
   - 统一了参数命名风格
