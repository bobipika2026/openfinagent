## 📋 问题反馈

### 问题 1: 策略无法回测
**用户反馈**: 创建策略后没有回测选项，策略一直是待回测状态

**原因**:
- 创建策略时没有自动回测选项
- 策略预览页面缺少回测按钮
- 用户无法触发回测流程

### 问题 2: 导航栏重复
**用户反馈**: Web 页面导航栏都重复了

**原因**:
- 每个页面都有自己的导航逻辑
- 缺少统一的导航框架
- 侧边栏内容重复

---

## ✅ 解决方案

### 1. 策略工厂增强

#### 添加自动回测选项
```python
# 在创建策略前，用户可以选择是否自动回测
auto_backtest = st.checkbox(
    "✅ 创建后自动回测",
    value=False,
    help="勾选后，策略创建完成后会自动运行回测"
)
```

**效果**:
- ✅ 创建策略时可勾选自动回测
- ✅ 按钮文字动态显示："🚀 创建策略" / "🚀 创建策略 + 回测"
- ✅ 创建后自动执行回测流程

#### 添加立即回测按钮
```python
# 在策略预览页面
if st.button("📊 立即回测", type="primary"):
    run_backtest_for_strategy(strategy)
```

**效果**:
- ✅ 策略预览时可随时点击回测
- ✅ 3 个操作按钮并排：编辑/回测/保存
- ✅ 清晰的回测进度提示

### 2. UI 导航框架重构

#### 统一的侧边栏导航
```python
def render_sidebar():
    with st.sidebar:
        # Logo 和标题
        st.markdown("## 📈 OpenFinAgent")
        
        # 主导航（6 个核心页面）
        page = st.radio(
            "导航",
            [
                "🏠 首页",
                "🤖 策略工厂",
                "📈 选股引擎",
                "🔔 监控中心",
                "📊 回测中心",
                "📚 策略库"
            ]
        )
        
        # 快捷操作
        st.button("➕ 创建策略")
        st.button("🔍 选股")
        st.button("🔔 新建监控")
```

**效果**:
- ✅ 统一的导航结构
- ✅ 清晰的页面分类
- ✅ 快捷操作按钮
- ✅ 去除重复内容
